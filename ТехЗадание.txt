Вести учёт:
	Кто вошёл (ФИО) + дата и время (Автоматически) + цель + запись в файл (Автоматически) +
	Кто вышел (ФИО) + дата и время (Автоматически) + запись в файл (Автоматически)+
Авторизация на рабочем месте:
	Время и дата (Автоматически) + ФИО + пароль и логин  + запись в файл (Автоматически)+
Все проверки и записи вести с текстовыми файлами.+
Название текстового файла должно содержать первые буквы ФИО смены + дата и время + подпись "note" в конце, с расширением .log+
Для лог файлов отдельная папка (Создаётся автоматически) +
Для учётных записей локальная папка (Создаётся автоматически) если папки нет. Если нет файла учётных данных, то вместо предложения авторизации, запрашивать конфигурационный файл. Файл будет в txt формате.+
Для каждой учётной записи отдельный лог файл.+
